### **4. Solucin de Problemas Comunes**

#### **A. Errores de Referencias Cruzadas Invlidas**

- **Mensaje de Error:**
  ```
  ValueError: Referencia cruzada invlida: CHXX no est en el mapeo de renumeracin ni existe en el documento.
  ```
  
- **Causa:**
  - Una referencia cruzada est apuntando a un ID que no existe en el mapeo de renumeracin o en los IDs existentes del documento.

- **Solucin:**
  - Revisa el documento JSON original para asegurarte de que todas las referencias cruzadas apuntan a captulos vlidos.
  - Actualiza el mapeo de renumeracin si es necesario.

#### **B. Captulos Duplicados**

- **Mensaje de Error:**
  ```
  ValueError: Duplicado detectado: ID de captulo CHXX ya ha sido asignado.
  ```

- **Causa:**
  - El mapeo de renumeracin est asignando el mismo nuevo ID a mltiples captulos.

- **Solucin:**
  - Revisa el diccionario `renumeracion_ids` para asegurarte de que cada antiguo ID se mapea a un nuevo ID nico.

#### **C. Falta de Exportacin de Algunas Secciones**

- **Causa:**
  - Si algunas secciones no tienen listas de captulos o tienen nombres de sub-secciones diferentes, pueden no exportarse correctamente.

- **Solucin:**
  - Asegrate de que todas las sub-secciones dentro de cada seccin contienen listas de captulos.
  - Ajusta el script si tienes sub-secciones con estructuras diferentes.
